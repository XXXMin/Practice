# 坦克大战小游戏（Python）— 需求文档与开发计划

## 1. 项目概述

使用 Python 实现一款单机 **2D 俯视角坦克大战** 小游戏：玩家操控坦克在矩形地图内移动、射击，消灭敌方坦克并保护基地（可选）或达成关卡目标。本文档描述 **需求范围** 与 **分阶段开发计划**，作为后续编码与验收的依据。

---

## 2. 需求文档

### 2.1 功能需求

| 编号 | 模块 | 需求说明 | 优先级 |
|------|------|----------|--------|
| F1 | 游戏窗口 | 固定分辨率或可配置窗口，游戏主循环（帧率稳定，如 60 FPS） | P0 |
| F2 | 地图 | 砖墙、钢墙、水域、草地（遮挡视野或仅装饰）、可破坏/不可破坏区分 | P0 |
| F3 | 玩家坦克 | 四方向移动、边界与墙体碰撞、一发或多发子弹规则（冷却） | P0 |
| F4 | 敌方坦克 | 简单 AI：随机或朝玩家方向移动、间歇射击；被击毁后消失或爆炸动画 | P0 |
| F5 | 子弹 | 直线飞行、与墙/坦克碰撞后消失；砖墙可被摧毁 | P0 |
| F6 | 生命与胜负 | 玩家生命数、全灭敌人胜利、玩家生命耗尽或基地被毁失败 | P1 |
| F7 | 基地 | 地图底部或固定位置基地，被敌方子弹击中则游戏失败 | P1 |
| F8 | UI | 开始界面、游戏中 HUD（得分/生命/关卡）、暂停、游戏结束与重开 | P1 |
| F9 | 关卡 | 至少一关可玩；关卡数据可用二维数组或文件描述，便于扩展 | P2 |
| F10 | 音效/特效 | 射击、爆炸、背景音乐（可选，可后续迭代） | P3 |

### 2.2 非功能需求

- **性能**：在常见办公本上流畅运行，无明显卡顿。
- **可维护性**：逻辑与资源（贴图、关卡数据）分离；核心玩法与渲染分层。
- **兼容性**：优先支持 Windows；依赖明确版本（如 `requirements.txt`）。

### 2.3 约束与假设

- **单机**：首版不实现网络对战。
- **操作**：键盘（方向键或 WASD + 射击键）；手柄为可选扩展。
- **技术**：推荐使用 **Pygame** 作为图形与输入框架（成熟、文档多）；若追求更轻量可评估 `arcade`，但本计划默认 Pygame。

### 2.4 验收标准（MVP）

- 可启动游戏，玩家坦克能在地图内移动、开炮。
- 至少一种敌方坦克能移动并射击，被击中后销毁。
- 墙体阻挡坦克，砖墙可被子弹破坏。
- 有明确胜利或失败条件（如全歼敌人或玩家阵亡）。
- 无致命崩溃；窗口可正常关闭。

## 3. 开发计划

### 3.1 阶段划分

| 阶段 | 名称 | 主要交付物 | 预估顺序 |
|------|------|------------|----------|
**阶段 0** | 工程搭建 | 虚拟环境、`requirements.txt`、入口 `main.py`、空窗口与游戏循环 | 1 |
**阶段 1** | 地图与碰撞 | 瓦片地图绘制、坦克与墙体的矩形碰撞 | 2 |
**阶段 2** | 坦克与子弹 | 玩家移动/转向、发射子弹、子弹与地图/单位碰撞 | 3 |
**阶段 3** | 敌人与 AI | 敌方生成、简易寻路/随机移动、自动射击、击毁判定 | 4 |
**阶段 4** | 规则与流程 | 生命、得分、胜负判定、基地（若采用）、关卡切换或重开 | 5 |
**阶段 5** | 体验打磨 | UI（菜单/HUD/暂停）、简单特效、音效（可选）、README 使用说明 | 6 |

### 3.2 里程碑

1. **M1 — 可运行原型**：窗口 + 玩家坦克在空白或简单背景上移动。
2. **M2 — 可战斗原型**：子弹、敌人、击毁、基础地图障碍。
3. **M3 — 可玩版本**：完整一局胜负 + HUD + 重开。
4. **M4 — 发布小版本**：关卡数据外置、依赖与运行说明齐全。

### 3.3 技术要点（实现时参考）

- **坐标系**：统一使用屏幕坐标（左上角为原点）或世界坐标 + 摄像机，全程一致。
- **碰撞检测**：AABB（轴对齐包围盒）即可满足俯视角坦克需求。
- **状态管理**：菜单 / 游戏中 / 暂停 / 结束 等状态用枚举或简单状态机管理。
- **时间步长**：使用固定逻辑帧或 `dt` 控制移动，避免帧率波动导致速度不一致。

### 3.4 风险与对策

| 风险 | 对策 |
|------|------|
| AI 过难或过弱 | 先参数化射速、移速，再调数值 |
| 碰撞边界 bug | 拆分「坦克-墙」「子弹-墙」「子弹-单位」测试用例 |
| 贴图版权 | MVP 使用简单几何图形或自绘像素块，后续再替换素材 |

### 3.5 后续扩展（非首版范围）

- 道具（护盾、加速、炸弹清屏）
- 多关卡与选关
- 本地双人
- 存档与最高分榜

## 5. 运行方式

1. 安装依赖（建议使用虚拟环境）：

   `pip install -r requirements.txt`

2. 在项目根目录 `XXX/Cursor` 下启动：

   `python main.py`

3. 操作说明：菜单中 **空格 / Enter** 开始；**WASD** 或 **方向键** 移动；**空格 / J** 射击；**ESC** 暂停；全歼敌方坦克且保护基地（金色方块）即胜利。

关卡数据默认读取 `src/levels/level01.json`；若文件缺失则使用 `level.py` 中的内置地图。
